

Добро пожаловать в документацию Yet Another Simple Scraper!
===========================================================


**Yet Another Simple Scraper** (Yass) представляет собой библиотеку по скрэпингу данных, максимально
облегчающую процесс сбора данных из Интернета для пользователей. Yass умеет работать по расписанию, 
использовать различные виды хранилищ

Основные особенности проекта:
-----------------------------

.. card-carousel:: 2

   .. card:: Все в одном месте
      
      
      В Yass имплементирован весь процесс:
      от определения ресурса до хранения информации.
   
   .. card:: Легко настраивать и использовать

      Приложение настраивается простым объявлением классов.
      Все нужные аттрибуты заполнятся на основании существующего
      контекста автоматически. Вся шаблонная логика скрэпинга уже реализована
      в Yass.

   .. card:: Высокая производительность

      Yass использует асинхронные функции, что значительно
      ускоряет опрос необходимых ресурсов.

   .. card:: Удобство использования

      Больше нет необходимости использовать множество разнородных
      инструментов - Yass подходит и для API, и для сырых веб-страниц.
